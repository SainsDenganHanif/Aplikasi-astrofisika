# Aplikasi astrofisika V 1.0
Sebuah aplikasi yang menggabungkan beberapa perhitungan untuk bintang dan planet

## FITUR-FITUR : 
- menghitung Kecerahan, suhu dan jari-jari bintang dari massanya

### FITUR FITUR YANG DIRENCANAKAN
1. **Menghitung properti bintang**
    - Memprediksi properti bintang dari 2 input, yakni massa dan umur bintang
    - Memprediksi properti bintang dari 3 input, yakni massa, umur bintang dan metalisitas

2. **Menghitung properti planet**
    - Menghitung zona layak huni disekitar planet
    - Menghitung orbit planet   
    - Menghitung stabilitas sebuah sistem bintang dan planet

## CARA KERJA APLIKASI INI
- Aplikasi ini menghitung properti bintang dari rumus kecocokan-empiris
